# 執行自訂報表

>[!NOTE]
>
>您也可以在 [建立](report-create.md) 或 [編輯](report-edit.md) 它。

1. 在主功能表中，按一下 **[!UICONTROL Reports]**.

1. 在 [!UICONTROL Options] 欄，按一下 **[!UICONTROL options]>[!UICONTROL Run Now]**.

1. 在確認訊息中，按一下 **[!UICONTROL OK]**.

   報表完成時，會傳送至報表設定中指定的所有電子郵件地址。
