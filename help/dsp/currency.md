# Adobe Advertising Cloud支援的貨幣

Advertising Cloud支援下列貨幣。

>[!NOTE]
>
>Adobe Analytics支援[較大的貨幣集](https://experienceleague.adobe.com/docs/analytics/admin/admin-tools/currency.html)。

| BRL | 巴西雷亞爾 |
| CAD | 加拿大元 |

| CNY | 人民幣 |
| DKK | 丹麥克朗 |
| 歐元 | 歐元 |
| GBP | 英鎊 |
| HKD | 港元 |

| JPY | 日元 |

| NZD | 紐西蘭元 |
| RUB | 俄羅斯盧布 |
| SGD | 新加坡元 |

>
